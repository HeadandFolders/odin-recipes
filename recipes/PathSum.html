Leetcode Path Sum Recipe

Description

given the root of a binary tree and a targetSum return true if a root-to-leaf path exists with sum=targetSum. Scrumptious!

Ingredients

  • helper function that takes 2 parameters
  • 2 if statements
  • 2 return statements

Steps

  1. in helper function, node and curSum are the parameters
  2. if statement: if not node, return False
  3. curSum+=node.val
  4. check if not node.left and not node.right, to then return whether curSum==targetSum
  5. return (helper function of .left OR .right)
  6. call helper function on root node
